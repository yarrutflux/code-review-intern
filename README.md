# Intern Exercises Repository

This repository contains three small front-end exercises implemented in React. Each exercise provides a dataset and an expected output. The goal is to complete the exercises so that the app renders correctly according to the given output.

## Exercises

### User Profile Card

Render a user profile based on a given dataset, including nested objects and arrays. Display the information exactly as shown in the expected output.

### Product Analytics Dashboard

Display product information and derived results using the dataset provided. Render the results clearly in JSX as shown in the expected output.

### Mini Todo App

Build a small interactive todo application where tasks can be added, toggled, and removed. Each task should be displayed according to the expected output.

## How to Use

1. Clone the repository.
2. Open an exercise in a React environment (e.g. CodeSandbox, StackBlitz, or locally).
3. Complete the exercise so the output matches the example provided.
4. Share your solution by pushing your branch or providing a sandbox link for review.
